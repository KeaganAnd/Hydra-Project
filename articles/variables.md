## Variables

#### What are variables?
Simply, variables are used to store data! It can be any number of things like numbers or text.

They can look like this

```lua
-- This is an integer
local x = 5

-- This is a string

local y = "Hello!"

-- This is a boolean
local z = false
local z2 = true

```
